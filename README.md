
# Daily Whisper Platform

A minimal, aesthetic daily quote web app with PWA, Google Sheets submission, favorites, and shareable images.

## Features
- Mobile-first (RWD)
- Smooth slide
- Random background images
- Hashtag filtering
- Favorites
- Google Sheets Submission
- Share images (IG Story)
- PWA (installable)

## Deploy
1. Upload to GitHub
2. Settings → Pages → root
3. Replace Google Sheets Web App URL in js/app.js
4. Done
